## 💾 СИСТЕМА ХРАНЕНИЯ

### todos.json

```json
{
  "12345": {
    "timezone": "Europe/Moscow",
    "todos": [
      {
        "id": 0,
        "task": "Купить молоко",
        "completed": false,
        "created_at": "2025-12-28T10:30:00",
        "reminder_time": "09:00"
      }
    ],
    "simple_todos": [
      {
        "id": 0,
        "task": "Позвонить маме",
        "completed": false,
        "created_at": "2025-12-28T10:35:00"
      }
    ]
  }
}
```

**Структура:**
- `timezone` - основной часовой пояс пользователя
- `todos` - задачи с напоминаниями (требуют reminder_time)
- `simple_todos` - быстрые дела без напоминаний (новое!)

## 📊 ТЕХНИЧЕСКАЯ ИНФОРМАЦИЯ

### Python версия
- Python 3.8+ (требует async/await)

### Основные зависимости
- `python-telegram-bot==20.7` - Работа с Telegram API
- `APScheduler==3.10.4` - Планирование напоминаний (новое!)
- `python-dotenv==1.0.0` - Управление переменными окружения
- `pytz==2024.1` - Работа с часовыми поясами

### Архитектура
- **Асинхронная** - все операции async/await
- **Модульная** - разделение на bot.py, database.py, config.py
- **Масштабируемая** - можно добавлять новые функции
- **JSON-based** - простая база данных в JSON

### Безопасность
- Токен хранится в `.env` (не в коде!)
- `gitignore` исключает `.env` и `todos.json`
- Нет hardcoded паролей или ключей
