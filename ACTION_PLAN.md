📖 ПОДРОБНЫЙ ПЛАН:

ЭТАП 1: ПОДГОТОВКА (3 минуты)
════════════════════════════════════════════════════════════════════════

Шаг 1.1: Получить токен от BotFather
──────────────────────────────────────

1. Открой Telegram
2. Найди @BotFather (поиск → @BotFather)
3. Отправь команду: /newbot
4. BotFather спросит имя бота
   └─ Напиши: MyCoolBot (или своё имя)
5. BotFather спросит username бота
   └─ Напиши: my_awesome_todo_bot (или своё)
6. BotFather отправит токен
   └─ Скопируй его (выглядит так: 123456:ABC-DEF...)

⏱️  ВРЕМЯ: 2 минуты

Шаг 1.2: Подготовить .env файл
──────────────────────────────

1. Открой терминал в папке проекта
2. Выполни команду:
   
   cp .env.example .env

3. Открой файл .env в редакторе (например VS Code)
4. Замени строку:
   
   TELEGRAM_BOT_TOKEN=your_bot_token_here
   
   на:
   
   TELEGRAM_BOT_TOKEN=твой_токен_который_ты_скопировал

5. Сохрани файл

⏱️  ВРЕМЯ: 1 минута

═══════════════════════════════════════════════════════════════════════

ЭТАП 2: УСТАНОВКА (1 минута)
════════════════════════════════════════════════════════════════════════

Шаг 2: Установить зависимости
────────────────────────────

1. Открой терминал
2. Перейди в папку проекта:
   
   cd /workspaces/reminder-tgbot

3. Установи зависимости:
   
   pip install -r requirements.txt

4. Дождись завершения

⏱️  ВРЕМЯ: 1 минута

═══════════════════════════════════════════════════════════════════════

ЭТАП 3: ЗАПУСК (1 минута)
════════════════════════════════════════════════════════════════════════

Шаг 3: Запустить бота
──────────────────

1. В терминале выполни:
   
   python3 bot.py

2. Если всё правильно, увидишь:
   
   🤖 Бот запущен!

3. Бот теперь работает и слушает твои команды!

⏱️  ВРЕМЯ: 1 минута

═══════════════════════════════════════════════════════════════════════

ЭТАП 4: ПРОВЕРКА (2 минуты)
════════════════════════════════════════════════════════════════════════

Шаг 4: Проверить бота в Telegram
──────────────────────────────

1. Открой Telegram
2. Найди своего бота
   └─ Используй username, который ты дал BotFather
3. Начни диалог с ботом
4. Отправь команду: /start
5. Ты должен увидеть:
   
   🤖 Добро пожаловать в To-Do Bot!
   [3 кнопки]

6. Нажми кнопку "➕ Добавить новую задачу"
7. Следуй инструкциям:
   
   Назови задачу:        Купить молоко
   Часовой пояс:         GMT+3
   Время напоминания:     09:00

8. Должно появиться сообщение:
   
   🎉 Отлично! Задача добавлена!

✅ ВСЁ РАБОТАЕТ!

❌ ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ:

1. ❓ "Бот не отвечает"
   └─ Проверь, что токен правильно добавлен в .env
   └─ Перезапусти бота
   └─ Проверь интернет соединение

2. ❓ "ModuleNotFoundError"
   └─ Установи зависимости: pip install -r requirements.txt
   └─ Проверь, что используешь правильное окружение

3. ❓ "Ошибка валидации времени"
   └─ Используй формат: ЧЧ:МММ (например: 09:00)
   └─ НЕ используй: 9:00, 14.30, 2:30 PM

4. ❓ "Ошибка валидации часового пояса"
   └─ Используй: GMT+3, GMT-5, GMT+5:30
   └─ НЕ используй: UTC+3, London, Asia/Moscow
