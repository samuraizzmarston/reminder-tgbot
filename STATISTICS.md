### Состояния (States) - 3

```
1. WAITING_TASK_NAME       - Ожидание названия задачи
2. WAITING_TIMEZONE        - Ожидание часового пояса
3. WAITING_REMINDER_TIME   - Ожидание времени напоминания
```

### Обработчики команд - 1

```
1. /start  - Главное меню с 3 кнопками
```

### Callback обработчики - 8

```
1. pending_tasks         - Показать активные задачи
2. completed_tasks       - Показать завершённые
3. add_task              - Начать добавление
4. complete_todo         - Отметить как завершённую
5. delete_todo           - Удалить задачу
6. back_to_main          - Вернуться в меню
7. cancel                - Отмена операции
8. back_to_main (fallback)
```

### Inline-кнопки - 10+

```
📋 Активные задачи
✅ Завершённые задачи
➕ Добавить новую задачу
◀️ Назад в меню
✓ Отметить как завершённую (динамические)
🗑️ Удалить задачу (динамические)
```

### Класс DatabaseTodo - методы

```
add_todo()              - Добавить задачу
get_pending_todos()     - Получить активные
get_completed_todos()   - Получить завершённые
complete_todo()         - Отметить как готовую
delete_todo()           - Удалить задачу
get_user_timezone()     - Получить часовой пояс
_load_data()            - Загрузить из JSON
_save_data()            - Сохранить в JSON
```
