# 🔄 МИГРАЦИЯ С GMT НА НОВУЮ СИСТЕМУ

## 📊 Что Изменилось?

### Старая Система (Версия <2.1)
```
Часовой пояс: GMT+4
Часовой пояс: GMT-5
Часовой пояс: GMT+5:30
```

### Новая Система (Версия 2.1+)
```
Часовой пояс: Asia/Baku (GMT+4)
Часовой пояс: America/New_York (GMT-5)
Часовой пояс: Asia/Kolkata (GMT+5:30)
```

---

## ✅ Преимущества Новой Системы

### 1. Поддержка Летнего/Зимнего Времени

**Старая система (GMT):**
```
GMT+3 - всегда одно и то же время
❌ Проблема: При переводе на летнее время вручную менять настройки
```

**Новая система (pytz):**
```
Europe/Moscow - автоматически учитывает переводы времени
✅ Преимущество: Всё работает автоматически!
```

### 2. Точность

**Старая система:**
```
GMT+5:30 - нужно помнить про полчасы
❌ Лёгко запутаться
```

**Новая система:**
```
Asia/Kolkata - точное название + часовой пояс
✅ Ясно и понятно!
```

### 3. Больше Выбора

**Старая система:**
```
Всего ~26 вариантов (GMT-12 до GMT+14)
```

**Новая система:**
```
Более 400 уникальных часовых поясов
```

## 🔧 Как Перейти?

### Шаг 1: Отметь Свою Старую Задачу

Если у тебя была задача с GMT:
```
Задача: Совещание в 14:00
Пояс: GMT+4
Напоминание: 14:00
```

### Шаг 2: Узнай Свой Новый Пояс

Твой пояс GMT+4?
```
✅ Это Asia/Baku (Баку, Азербайджан)
или Asia/Tashkent (Ташкент, Узбекистан)
или другой город на GMT+4
```

### Шаг 3: Добавь Новую Задачу

```
/start → ➕ Добавить новую задачу

Шаг 1: Совещание в 14:00
Шаг 2: Выбери Asia/Baku (вместо GMT+4)
Шаг 3: 14:00

✅ Готово!
```

### Шаг 4: Удали Старую Задачу

Если у тебя были старые GMT задачи, удали их:
```
✅ Завершённые задачи → 🗑️ Удалить старую задачу
```

## 🌍 Как Определить Свой Часовой Пояс?

### Способ 1: По Времени

Если сейчас у тебя 15:00, а в Лондоне 11:00:
```
15:00 - 11:00 = 4 часа
✅ Твой пояс: GMT+4 → Asia/Baku
```

### Способ 2: По Городу

```
Я в Москве → Europe/Moscow (GMT+3)
Я в Баку → Asia/Baku (GMT+4)
Я в Нью-Йорке → America/New_York (GMT-5)
Я в Токио → Asia/Tokyo (GMT+9)
```

### Способ 3: Спросить у Бота

```
Пауза в выборе часового пояса
→ Нажми "📌 Все часовые пояса"
→ Введи название своего города

Примеры:
Europe/Moscow
Asia/Baku
America/New_York
```

## ❓ Часто Задаваемые Вопросы

### Q: Где найти полный список всех поясов?

**A:** В боте нажми "📌 Все часовые пояса" при добавлении задачи, или смотри:
https://en.wikipedia.org/wiki/List_of_tz_database_time_zones

### Q: Мой пояс GMT+5:45, что делать?

**A:** Введи в боте "Asia/Kathmandu" - это Непал на GMT+5:45

### Q: Я был на GMT+4, а потом переехал на GMT+2, нужно менять все задачи?

**A:** Нет! Напоминания используют сохранённый пояс каждой задачи. Просто обновляй часовой пояс в новых задачах.

### Q: Будут ли мои старые GMT задачи переведены автоматически?

**A:** К сожалению нет, но ты можешь удалить старые и создать новые (это займёт 2 минуты).

### Q: Я не помню свой точный часовой пояс, что делать?

**A:** Найди свой город в списке:
- При добавлении задачи нажми "📌 Все часовые пояса"
- Введи свой город (например: Moscow, Tokyo, New_York)
